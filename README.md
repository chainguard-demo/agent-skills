# Chainguard Agent Skills — Technical Demo

The engineering walkthrough for an Agent Skills demo. Run the commands from this folder.

- `demo.md` — the script. "Short version" is the tl;dr; "Full script" is the narrated version.
- `workstation-checkup-sample-skill/` — the clean reference skill, also used in the publish step.
- `malicious-skill-example/` — real, defanged bad-skill samples (`fun-brainstorming`, `find-skills`). Read them, don't run them.
- `hardening-comparison/web-design-upstream/` — the useful-but-vulnerable skill as published upstream.
- `hardening-comparison/web-design-hardened/` — the same skill after Chainguard hardening, with its `HARDENING.md` report. Diff the two to see what changed.
